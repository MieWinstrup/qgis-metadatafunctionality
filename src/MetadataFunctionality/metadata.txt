about=
    <img src="http://www.septima.dk/skin1/img/logo_143x39.png" alt="Septima">
    <p>MetaMan stores metadata info on any PostgreSQL table.</p>

    In order to get started you have to create a table in one of your PostgreSQL databases.

    <br/>
    <i>
    DROP TABLE IF EXISTS "metaman";
    CREATE TABLE "metaman" (
        "resp_center_off" varchar COLLATE "default",
        "beskrivelse" varchar COLLATE "default",
        "name" varchar COLLATE "default",
        "timestamp" varchar COLLATE "default",
        "proj_wor" varchar COLLATE "default",
        "table" varchar COLLATE "default",
        "journal_nr" varchar COLLATE "default",
        "guid" varchar COLLATE "default",
        "db" varchar COLLATE "default",
        "schema" varchar COLLATE "default"
    )
    WITH (OIDS=FALSE);
    ALTER TABLE "metaman" OWNER TO "metaman";
    </i>

    <br/>
    Then configure MetaMan: navigate to the table, enter username and password and you are set to go.
